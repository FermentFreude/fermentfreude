# UI Pattern Registry

A lookup index of reusable UI/component decisions, captured by `/project:imprint` after building something new. Not documentation of how the code works — `DESIGN_SYSTEM.md` covers tokens, `AGENTS.md` covers Payload patterns. This file is for the judgment calls: "we solved X this way, reuse it instead of inventing a new approach."

Each entry should stay under ~5 lines. Delete an entry if the pattern it describes gets replaced.
