What Webflow still couldn't do — even with all those tools

  • Custom seat selection per booking date (e.g. "2 of 8 spots left — book 2")
  • Workshop appointment management directly in the admin — no Webflow tool replicates this exactly
  • Voucher system with unique codes, redemption flow, and booking link
  • One cart for workshops + shop products in a single checkout
  • Custom admin dashboard tailored to David and Marcel's workflow
  • Full control over design, fonts, animations — no Webflow template constraints

These features would require a Webflow developer to build custom workarounds — additional one-time cost on top of the monthly subscriptions.

Bottom line

  • A comparable Webflow setup would cost ~€285/month — €3,420/year — and still wouldn't match all current features.
  • The custom build has almost no recurring SaaS costs. The main cost going forward is developer time for maintenance and new features.
  • A developer retainer significantly below €285/month already makes the custom build the better financial decision — and the product is better.
  • The website, code, and all content are owned by FermentFreude. There is no platform lock-in — unlike Webflow, where leaving means rebuilding from scratch.
