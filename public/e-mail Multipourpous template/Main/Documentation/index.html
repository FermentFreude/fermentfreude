B) Templates - top

Udos has set of 11 Responsive Htmls which can be use for any kind of business like Travel, Retail, Food, Education, Product Promotion, Healthcare & B2B and even much more. Also Included Thank you Email Templates which can be use for Order Summary, Thankyou mail, View Order Status etc.

These templates contain all the layouts and you can easily edit or modify the content and images. The templates follow a new and easy to edit structure with proper comments in place.

C) Html Structure - top

  1. All Email templates follow a unique html structure which has been tested on browser & its responsive html are ready to use on all the majaor Campaign Monitor.
  2. Every module/block is nested inside a table and the same struture appliess for all available modules.
  3. All modules are marked with comments a comment is place at the start of a modules and is also marked at the end of the module.

The image blow will give you a clear idea of the html structure and the commenting.
